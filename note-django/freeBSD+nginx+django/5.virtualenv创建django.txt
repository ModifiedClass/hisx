如果此时报错，提示oduleNotFoundError: No module named '_sqlite3'
https://pkgs.org/download/sqlite找到适合的版本连接下载安装sqlite3
# pkg install py36-sqlite3
安装成功后就可以创建Django app了，Django开发环境搭建成功！

修改settings.py

加入
sys.path.insert(0,os.path.join(BASE_DIR,'apps')) 
新创建app可以(ENV1)# python3 manage.py startapp app名称 apps 

把生成的app名称加入INSTALLED_APPS序列中
NSTALLED_APPS=[
    ...,
    'rest_framework',
    'rest_framework.authtoken',
    app名称,
    ...,
]

MIDDLEWARE 中注释
    #'django.contrib.sessions.middleware.SessionMiddleware',
    #'django.contrib.auth.middleware.AuthenticationMiddleware',
    #'django.contrib.messages.middleware.MessageMiddleware',

域名访问权限
ALLOWED_HOSTS=[]修改为ALLOWED_HOSTS=['*']

设置模板路径
TEMPLATES=[
    ...
    'DIRS':[os.path.join(BASE_DIR,'templates')],
    ...
]

设置时区和中文
LANGUAGE_CODE='zh-hans'
TIME_ZONE='Asia/Shanghai'

设置静态文件
STATIC_URL='static'
STATICFILES_DIR=[
    os.path.join(BASE_DIR,'static'),
]
STATIC_ROOT=os.path.join(BASE_DIR,'static')
